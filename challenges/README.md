# Hello World React Application

Welcome to the Hello World React application! This is a basic React application that demonstrates how to create a simple component and display it in the browser.

## Getting Started

To get started with the application, follow these steps:
<!-- Run this in your commandline -->
1. npm Create vite@latest
2.Insert your Project Name at the prompt <Project Name>
3.Choose the framework your working with - React
4. Choose the language your working with - Javascript
5. Cd into your Project directory
6. Install the required dependencies by running the command `npm install` in the project directory.
y. Start the development server by running the command `npm  run dev`.
7. Open your browser and navigate to the localhost to see the application in action.

## Project Structure

The project has the following structure:

- `App.jsx`: The main component that serves as the entry point for the application. It renders the heading, paragraph, and the `Hello` component.
- `App.css`: A CSS file that contains the styling for the application.
- `components/Hello.jsx`: A simple React component that displays a greeting. It receives a `name` prop and renders a heading with the name.
